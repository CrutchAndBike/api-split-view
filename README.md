# API для проекта "Split View"

## Требования для запуска:

1. Нужна установленная и запущенная MongoDB (https://www.mongodb.com).
1. Node.js (v10.12.0)

 ## Настройка MongoDB:
 
 Для упрощения разработки на данный момент не требуется.
 
## Запуск:

Dev-режим:
```bash
npm run dev
```

Запуск prod-режима:
```bash
npm run pro
```

Остановка prod-режима:
```bash
npm run prostop
```
